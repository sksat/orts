# orts-plugin-sdk

SDK for writing orts WASM plugin guests (callback and main-loop styles).

Provides procedural macros and helpers that the `cargo component`-built
guest crates consume to expose attitude / orbital controllers to the
[orts](https://github.com/sksat/orts) simulation host runtime via the
WebAssembly Component Model.

No runtime dependencies — the macros generate code that references the
consumer crate's `bindings` module (generated by `cargo component` from the
`orts.wit` interface definition).

## Usage

See the example plugin crates under [`plugins/`](../) in the orts
workspace for concrete usage:

- `orts-example-plugin-bdot-finite-diff`
- `orts-example-plugin-pd-rw-control`
- `orts-example-plugin-pd-rw-unloading`
- `orts-example-plugin-detumble-nadir`
