orts の WASM Component Model plugin runtime を使い、姿勢制御コントローラを
guest plugin として実装する example 群。コールバック型とメインループ型の
2 つの API スタイルを示す。

## Callback style (Plugin trait)

`orts-plugin-sdk` の `Plugin` derive で `on_tick` コールバックを実装。
ホストが tick ごとに呼び出す。

| Plugin | Description |
|---|---|
| **pd-rw-control** | PD 姿勢制御 + リアクションホイール出力 |
| **pd-rw-unloading** | PD 姿勢制御 + 磁気トルカによる RW アンローディング |
| **detumble-nadir** | Detumble → nadir pointing モード遷移デモ |

## Main-loop style (wait_tick)

`wait_tick()` でホストから tick を受け取るメインループ型。

| Plugin | Description |
|---|---|
| **bdot-finite-diff** | B-dot 有限差分法 detumbling controller |

## Running

```sh
# Build a plugin
cd plugins/pd-rw-control
cargo component build --release

# Run simulation with plugin
cargo run --bin orts -- run \
  --config examples/pd-rw.toml \
  --plugin target/wasm32-wasip1/release/orts_example_plugin_pd_rw_control.wasm
```
